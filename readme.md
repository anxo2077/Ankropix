# Ankropix Website (HTML/CSS)

Repository: https://github.com/anxo2077/Ankropix  
page: https://anxo2077.github.io/Ankropix/  
Simple static website project built with HTML and CSS.  
It represents Ankropix, a fictional hosting service for gamers.

## Description

This project shows how to create a basic website using HTML and CSS.  
All pages share a consistent navigation bar and design.

Includes:

- Home page with a short introduction to Ankropix.
- Services page describing hosting options.
- Pricing page with simple plan cards.
- About page with basic information about the project.
- Contact page with a basic form (no backend).

## Requirements

- Any modern web browser (Chrome, Firefox, Edge, etc.)
- Files saved in the correct structure

No JavaScript, database, or server is required.

## How to Open

1. Download or clone this repository.
2. Make sure the files are in the same folder (or correct paths).
3. Open `index.html` in your browser or usin the page.
4. Use the top navigation menu to move between pages.

## Project Structure

- `index.html`  
  Main landing page.

- `services.html`  
  Lists services offered by Ankropix.

- `pricing.html`  
  Shows example plans and prices.

- `about.html`  
  Explains what Ankropix is and its purpose.

- `contact.html`  
  Simple contact form (name, email, message).

- `styles.css`  
  Handles layout, colors, fonts, buttons, and sections.

## Features

- Basic navigation.
- Simple and clean.
- Reusable header and footer.
- Basic typography and color scheme.
